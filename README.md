# Hospital Asset Management and Tracking Portal

Django-based hospital asset management system for tracking, monitoring, and managing medical equipment.

## Project Report
Detailed documentation is available in the uploaded PDF report:
**S6 Miniproject by Tessa Jose.pdf**

## Technologies Used
- Python
- Django
- SQLite
- HTML
- CSS
- JavaScript
- Bootstrap

## Features
- Secure User Authentication
- Role-Based Access Control
- Asset Registration
- Asset Tracking
- Asset Location Management
- Dashboard Monitoring
- Search Functionality

- QR/Barcode Integration
